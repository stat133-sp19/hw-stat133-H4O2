Binomial

The package "binomial" is an implementation for the basic statistics for a random from binomial distribution, and to visualize the relative plots (probability and cumulative distribution).

Creating a binvar

The first step is to create a "binvar" object with the function bin_variable():

binvar <- bin_variable(10, 0.3)
binvar
#> [1] "Binomial Variable"
#> [1] "Parameter"
#> [1] "- number of trials: 10"
#> [1] "- prob of success: 0.3"

bin_variable() creates a binomial variable with trials and prob as input and its type is the "binvar", that is, a binomial random variable object.

The "binvar" have 3 methods, like prinr,summary,print.summary where you can check the relaetd statistics (e.g.Â mean, variance, kurtosis). For example, the summary() function gives you a nice table of statistics:

summary(binvar)

Show the probability

In this package, you can use the function bin_probablity() to get the probablity of one or more success given number of trials. Or the bin_choose() to get the number of possible combinations

bin_choose(5, 3)
#> [1] 10
bin_probability(1:5, 10, 0.3)
#> [1] 0.1210608 0.2334744 0.2668279 0.2001209 0.1029193

Plot of relative distributions and cumulatives

The output of bin_distribution() will be an object of class "bindis", which will contain the data.frame of success number and its corresponding probability with method plot:

# plotting binomial probability distribution
dis1 <- bin_distribution(trials = 5, prob = 0.5)
dis1
#>   success probability
#> 1       0     0.03125
#> 2       1     0.15625
#> 3       2     0.31250
#> 4       3     0.31250
#> 5       4     0.15625
#> 6       5     0.03125

You can use the function plot() to create a graphic:

plot(dis1)

The output of bin_cumulative() will be an object of class "bindis", which will contain the data.frame of success number and its corresponding probability as well as cumulative probability with method plot:

# plotting binomial probability distribution
dis2 <- bin_cumulative(trials = 5, prob = 0.5)
dis2
#>   success probability cumulative
#> 1       0     0.03125    0.03125
#> 2       1     0.15625    0.18750
#> 3       2     0.31250    0.50000
#> 4       3     0.31250    0.81250
#> 5       4     0.15625    0.96875
#> 6       5     0.03125    1.00000

You can use the function plot() to create a graphic

plot(dis2)
